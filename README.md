# Nest Watcha

http://localhost:3000/api/

Used TypeScript , Nest js , MySQL , Docker , CI CD

# 📌 구현한 기능

## 👉 회원가입 & 로그인(Signup & Signin)

- bcrypt 비밀번호 암호화
- Session , cookie 로그인

## 👉 게시글

- Board (CRUD)
- Image Upload
- HashTa

## 👉 댓글

- Comment (CRUD)

## 👉 장르

- 어드민 Permission 체크
- 장르 ( CRUD )

## 👉 영화

- 어드민 Permission 체크
- 뮤비 ( CRUD ) 

## 👉 주문

- Order iamport
